# 自动化生成项目模版

## 框架说明

react + flux + antd + webpack + es6

## 功能说明

1. 能通过接口多次声称一下基本的代码
	* 能生成基本的curd，
	* 能生成列表的表头，
	* 能生成搜索条件的数据，
	* 能生成提交的表单域
	* 能生成当前项目的各模块的目录结构
2. 能根据模块的目录结构自动生成基本的模块代码，包含基本的curd功能
3. 能指定一个目录生成基本的curd功能

## 生成内容说明

1. 每个独立接口的文件
2. 每个模块中的接口请求，包含基本的curd接口请求，并以约定的名称命名请求
3. 每个模块中列表的表头
4. 每个模块中的搜索条件
5. 每个模块中的表单

## 接口定义约定（管理后台）

1. 只使用两级路由，pintuan/search pintuan/:id
